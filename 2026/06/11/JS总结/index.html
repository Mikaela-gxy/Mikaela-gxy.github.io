JavaScript总结

一、JS 代码引入 HTML 的两种方式

1. 内嵌式（写在 HTML 内部）

  • 使用 <script></script> 标签包裹 JS 代码
  • 规范写在 <body> 标签末尾
  • 旧写法：<script type="text/javascript" charset="utf-8">
  • 现代浏览器可省略全部属性，直接写 <script>
  • 注意：仅 HTML 中需要 script 标签，独立 .js 外部文件不能写 script 标签

2. 外部引入式（推荐，代码分离）

  • JS 代码单独存为 .js 文件
  • 通过带 src 属性的 script 标签引入

优势：HTML、JS 代码分离，项目易维护。

三、字符串相关语法

1. 普通字符串拼接

使用 + 加号拼接字符串与变量，写法繁琐。

2. 模版字符串（反引号 ` 包裹，推荐）

标识：反引号 ` 替代单 / 双引号

变量插入：占位符 ${变量名}，无需加号拼接

支持原生换行：普通字符串换行必须加转义符 \n

3. 转义符 \

用于输出和引号、反引号冲突的符号：

转义符 说明
\" 双引号字符串内输出双引号
`` 模版字符串内输出反引号
\n 普通字符串换行

四、变量声明 let / const

1. let 规则

  • 可只声明不赋值，默认值为 undefined
  • 同一作用域禁止重复声明同名变量，会报语法错误
  • 声明后支持多次重新赋值

2. const 规则

  • 声明时必须同步赋予初始值，缺省值直接报错
  • 仅能赋值一次，后续重新赋值会报错
  • 同作用域不可重复声明同名变量

let、const 对比

特性 let const
声明时赋值 可选 必须
重新赋值 允许 禁止
重复声明 禁止 禁止

2. NaN（Not a Number，非数值）

含义：数学运算执行失败，但数据类型仍为 number

产生场景：

  • 非数字字符串做 - * / 运算（如 "abc" / 10）
  • 0 / 0
  • NaN 和任意数字运算，结果依旧是 NaN

- * / 非加法运算：数字字符串自动转为数字再计算

七、相等判断运算符

运算符 说明
== 宽松相等 仅对比值，会自动隐式转换类型
=== 严格全等 先判断数据类型是否一致，类型相同再对比值

推荐：开发优先使用 ===，避免类型转换 bug。

八、数据转布尔值规则

真值（转换后为 true）

  • 字符串：所有非空字符串
  • 数字：所有非 0 数字

假值（转换后为 false）

类型 值
字符串 空字符串 ""
数字 数字 0

九、数组基础增删改查操作

（一）增（添加元素）

方法 说明
push(元素1, 元素2...) 数组尾部添加元素，支持一次性添加多个
unshift(元素1, 元素2...) 数组头部添加元素，用法同 push

（二）删（删除元素）

方法 说明
pop() 无参数，删除数组最后一个元素，返回被删除项
shift() 无参数，删除数组第一个元素，返回被删除项
splice(起始下标) 从指定下标删除至数组末尾
splice(起始下标, 删除个数) 从指定下标删除对应数量元素

（三）改（修改指定位置元素）

语法：splice(起始下标, 删除数量, 替换新元素)

（四）查（查找元素下标 indexOf()）

单个参数（常用）：indexOf(查找值)

  • 找到元素：返回对应下标
  • 未找到：返回 -1

两个参数：indexOf(查找值, 起始搜索下标)

  • 从第二个参数指定下标开始向后检索
  • 找不到返回 -1
