C语言总结

基础概念

补码计算： 反码 + 1 = 补码

格式化输出

printf 常用格式符

格式符 说明
%.数字 ① 整形 = 显示指定位数，不够用 0 在前面补上
② 浮点 = 小数点后显示指定位数，不够用 0 在后面补上
%数字 若宽度更大则不做处理，若更小则在前面用空格补齐（若使用 %0数字，则用 0 代替空格）
%- 使产生的内容左对齐（使第一个占位符 %数字 补齐的空格失效，后面的占位符不受影响）
%+ 使产生的内容总带有符号（正数内容总带有 +，负数内容总带有 -）
%# 八进制前加 0，十六进制前加 0X

运算符

正负号

若 int a = -100：

  • +a = -100（保持原符号）
  • -a = 100（符号取反）

自增自减

模式 说明
++a / --a 前缀模式：先改变后使用
a++ / a-- 后缀模式：先使用后改变

运算符 说明 示例
|| 或（注意：a==1||2 是错的，应是 a==1||a==2） 6||0 = 1
&& 与 6&&0 = 0，6&&1 = 1

逻辑运算规则：

  • 非 0 值 = 1 为真
  • 0 = 0 为假

循环语句

while 循环

for 循环

说明：

  • for 循环可单独拿出一步执行但 ; 不能删去
  • for 循环可省略条件即为死循环，或将其设为真

do-while 循环

注意：

  • do while 的 while 后要加 ;
  • 至少循环一次

switch 语句

执行规则：

  • 找到对应的 case 后从此处 case 往下执行
  • 如无 break 则会一直往下执行完毕

数组

基本规则

  • 数组下标从 0 开始，即 0 对应第一个元素
  • 元素短于数组的长度，则后面用 0 填补（字符数组将 0 视为 \0 正常结尾）
  • 若更长则无法编译通过（字符数组因 \0 占用一个，所以相等也无法编译通过）

字符数组

  • 字符数组会在末尾加上 \0 标记结束
  • 但 \0 占用一个元素位置
  • 若有 10 个字母，则会视为共有 11 个元素

指针与二维数组

二维数组与指针

先明确两个基础定义（以 3×3 矩阵为例）：

场景 1：二维数组 + 数组指针（双层解引用）

场景 2：一维数组模拟二维矩阵 + 普通 int 指针（单层解引用）

终极等价关系总结

最简记忆口诀：

  • 数组指针（指向行）： 双层解引用，直接用 p[i][j]（和普通数组一样）
  • 普通指针（指向元素）： 单层解引用，算偏移 i × 列数 + j

指针传参

指针传参可以修改实参的值：

  • 普通传参是 “值传递”（传递的是变量的副本），函数内修改副本不会影响原变量
  • 指针传参是 “地址传递”，函数内通过解引用指针，直接操作原变量的内存，因此能修改原变量的值
