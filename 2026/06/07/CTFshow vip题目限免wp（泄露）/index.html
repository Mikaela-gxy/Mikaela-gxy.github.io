robots后台泄露

Robots 协议（Robots Exclusion Protocol）是一种网站与网络爬虫之间的“君子协定”，通过在站点根目录放置 robots.txt 文件，告知爬虫哪些路径可以抓取、哪些禁止访问，从而保护隐私、优化抓取效率、减少服务器压力。

vim临时文件泄露

题目告诉了vim临时文件泄露

常见的 Vim 临时文件类型：

文件类型 说明
.文件名.swp 正在编辑文件时生成的交换文件（核心）
.文件名.swo 多次编辑同一文件时生成的旧交换文件
.文件名.swn 更早的交换文件（以此类推）
文件名~ 编辑后保存的备份文件

主要是swp

扫下swp(注意这里dirsearch会自动加默认后缀php，可以指定后缀php.swp能扫出来，CTF 中涉及 web 类题目（占比最高）的核心是PHP 网站漏洞，出题方会故意留下 Vim 临时文件（.swp），而这些临时文件对应的原文件几乎都是 PHP 文件,故可扫php.swp)

但一般提示vim泄露直接访问index.php.swp更方便快捷

这里也是扫出了index.php.swp

访问得到一个文件

打开得到flag

cookie泄露

则输入echo “ctfshow%7B6a4d40a0-4075-40a3-a5a5-18b7fcb7ef79%7D” | python3 -c “import sys, urllib.parse; print(urllib.parse.unquote(sys.stdin.read().strip()))”
