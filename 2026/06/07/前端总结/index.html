HTML与CSS前端基础知识点归纳

一、HTML基础知识

1. 基础标签与骨架

  • 单标签：
    • ① <br> (换行)
    • ② <hr> (水平线)
  • 基本骨架：输入 !（英文）+ 回车或 tab 自动生成 HTML 基本骨架
  • 标题标签：<h1> 到 <h6>
  • 段落标签：<p>
  • 快捷键：Alt + Z (折行)，Ctrl + / (一键注释)

2. 文本格式化标签

效果 标签一 (推荐语义化) 标签二
加粗 <strong> <b>
倾斜 <em> <i>
下划线 <ins> <u>
删除线 <del> <s>

3. 核心媒体与链接标签

  • 图片标签：<img src="">
    • alt: 替换文本（图片无法显示时显示）
    • title: 提示文本（鼠标悬停时显示）
    • width / height: 宽度和高度（值为数字，无单位）
  • 超链接：<a href="">网页显示的文字</a>
    • href: 跳转地址（必须属性）。开发初期不确定时可写 #
    • target=”_blank”: 实现新窗口打开页面
  • 视频标签：<video src="">
    • 常用属性：controls (显示控制面板), loop (循环播放), muted (静音播放), autoplay (自动播放，为了体验通常需配合 muted 使用)
  • 音频标签：<audio src=""> (支持格式：MP3, Ogg, Wav，属性与视频类似)

4. 列表与表格标签

  • 无序/有序/定义列表：
    • <ul> 嵌套 <li> (无序列表)
    • <ol> 嵌套 <li> (有序列表)
    • <dl> 嵌套 <dt> (标题) 和 <dd> (详情描述)
  • 表格基本结构：<table> 嵌套 <tr> (行)，行嵌套 <th> (表头单元格) 或 <td> (内容单元格)
  • 合并单元格：保留最左、最上单元格，添加 rowspan (跨行合并) 或 colspan (跨列合并) 属性，并删除其他被合并的单元格

5. 表单标签

  • Input 标签分类：
    • text (文本框), password (密码框)。添加 placeholder 可实现提示文字
    • radio (单选框): 同组需 name 相同。默认选中加 checked
    • checkbox (多选框): 默认选中加 checked
    • file (上传文件): 默认单文件，加 multiple 可多选
  • 下拉菜单：<select> 嵌套 <option> (加 selected 默认选中)
  • 文本域：<textarea> (多行文本)
  • 按钮：<button type="submit|reset|button"></button>
  • label 标签 (增大点击范围)：
    • 写法一：<label for="id">文字</label>（要求 for 属性等于表单控件的 id）
    • 写法二：<label><input type="...">文字</label>

6. 字符实体

  • 空格：&nbsp;
  • 小于号 ( < )：&lt;
  • 大于号 ( > )：&gt;

二、CSS基础知识

1. CSS引入与选择器

  • 引入方式：内部样式表 (<style> 标签)、外部样式表 (<link> 引入)、行内样式表 (style=”…” 属性)
  • 基础选择器：标签选择器、类选择器 (.类名)、id 选择器 (#id名)、通配符选择器 (*)
  • 复合选择器：
    • 后代选择器 (空格)、子代选择器 (>)、并集选择器 (,)、交集选择器 (连写)
    • 伪类选择器：超链接四个状态 :link, :visited, :hover (悬停), :active
  • 结构与伪元素：:first-child, :nth-child(n) (公式如 2n 偶数，2n+1 奇数) 等；::before 和 ::after (需配合 content:”” 使用)
  • CSS 特性：继承性、层叠性 (相同覆盖，不同叠加)、优先级计算机制

2. 字体与文本属性

属性名 功能描述
font-size 字体大小 (单位：px)
font-weight 字体粗细 (normal/400, bold/700)
font-style 字体样式 (normal, italic 倾斜)
line-height 行高。单行文字垂直居中：高度 = 行高
text-indent 文本缩进 (推荐单位：em，1em = 当前字号大小)
text-align 文本水平对齐方式 (left, center, right)
text-decoration 文本修饰线 (none 无, underline 下划线, line-through 删除线)

3. 显示模式与背景

  • 显示模式 (display)：
    • 块级元素 (独占一行，可设宽高)、行内元素 (一行多个，宽高不生效)、行内块元素 (一行多个，可设宽高)
  • 背景属性：
    • background-image: url(路径)
    • background-repeat: (no-repeat, repeat-x, repeat-y)
    • background-position: 背景定位 (center, left 等，或具体坐标数值)
    • background-size: 背景缩放 (cover 比例缩放完全覆盖, contain 比例缩放完全装入)
    • background-attachment: fixed (背景固定不随内容滚动)

4. 盒子模型

  • 盒子尺寸计算：内容尺寸 + padding (内边距) + border (边框)。给盒子加 padding 或 border 默认会撑大盒子
  • 内减模式：box-sizing: border-box; (手动减去尺寸，解决撑大问题)
  • 清除默认样式：* { margin: 0; padding: 0; }
  • 外边距问题 (margin)：
    • 塌陷问题 (父子级)：给父级设置 overflow: hidden 或 border-top，或取消子级 margin 改父级 padding
    • 合并现象 (垂直的兄弟级)：上下 margin 会产生合并现象，取较大值生效
  • 元素溢出控制 (overflow)：hidden (溢出隐藏), scroll (显示滚动条), auto (自动)
  • 圆角与阴影：border-radius 设置外边框为圆角；box-shadow 设置盒子阴影

5. 浮动与Flex布局

  • 浮动 (float)：脱标、顶对齐。浮动元素会脱离标准流
    • 清除浮动 (解决父级高度为 0 导致塌陷)：额外标签法 (clear:both)、单/双伪元素法 (.clearfix::after)、父元素添加 overflow:hidden
  • Flex 布局：
    • 创建容器：给父级设置 display: flex;
    • 修改主轴方向：flex-direction (row 默认水平, column 垂直)
    • 主轴对齐方式：justify-content (flex-start, center, space-between, space-around, space-evenly 等)
    • 侧轴对齐方式：align-items (单行), align-content (多行)
    • 弹性伸缩比：flex (整数数字，表示占用剩余尺寸的份数)
    • 弹性盒子换行：flex-wrap: wrap;

6. 网页制作技巧

  • 项目目录：包含 images(图片), css(样式表), uploads(上传内容), index.html 等
  • 导航制作 (nav)：推荐标签结构 ul > li*n > a 避免单纯堆砌 a 标签以防搜索排名降级
  • Logo 技巧：推荐结构 h1 > a > 网站名称 (提升 SEO)。利用 CSS 背景图展示 Logo，并通过 font-size: 0; 隐藏文字

7. 定位属性 (Position)

五个定位属性

属性 说明
static 默认值，正常文档流排列
relative 相对定位，不脱标
absolute 绝对定位，脱标悬浮
fixed 固定定位，相对视口
sticky 粘性定位，滚动吸顶

relative（相对定位）

  • 不脱标，保留原位置占位
  • 相对自身原位置移动
  • 最常用场景：给 absolute 作父容器

absolute（绝对定位）⭐最常用

  • 脱标，不占位，悬浮在页面上方
  • 口诀：子绝父相
  • 相对最近带定位的父级定位（若无则相对文档根元素）

居中公式：

fixed（固定定位）

  • 脱标，不占位
  • 相对浏览器视口定位
  • 滚动屏幕时位置不变（常用于返回顶部按钮、侧边栏）

sticky（粘性定位）

  • 滚动到指定位置后固定
  • 兼具 relative 和 fixed 的特点

定位通用规则

  • 方向属性：可用 top / left / right / bottom 控制位置
  • 层级控制：z-index: 999;（数值越大越在上层）

最简记忆口诀：

  • relative 当爹（给绝对定位元素当容器）
  • absolute 悬浮（子绝父相）
  • fixed 钉死屏幕
  • sticky 滚动吸顶
