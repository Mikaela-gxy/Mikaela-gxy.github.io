字符串操作

常用方法

方法 说明
.length() 获取字符串长度
.charAt(n) 取出第 n 个字符
.contains(字符串) 判断是否包含指定字符串
.trim() 去除字符串前后的空格
.indexOf("字符串") 查找字符串首次出现的位置，返回 -1 表示没找到
.indexOf("字符串","开始索引值") 从指定索引值位置开始查找
.substring(n) 截取第 n 个字符后的字符串，不包括 n
.substring(n,n+m) 截取第 n 个字符后的 m 个字符的字符串
.endsWith("字符串") 结尾判断，返回布尔值
.startsWith("字符串") 开头判断，同理
.replaceAll("要替换的值","新值") 替换所有匹配的字符串
.split("分割字符串") 以分割字符串为分界线分割，每一块 = 数组的一个元素
.toUpperCase() 全大写
.toLowerCase() 全小写
.equals("字符串") 字符串比较，比 == 更规范

数据处理规则

  • 取出来本来就是字符串 → 可加可不加

  • 看见 [ ] 中括号 → 用 List

  • Map 按键取值 → 用名字 get(“key”)
